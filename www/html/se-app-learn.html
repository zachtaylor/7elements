Learn More About 7Elements

7Elements is a brand new card game, designed to simplify the format
You can play Spells, summon Bodys and Items, and attack and use your special powers

Website

This app streams data, for speed and to save on bandwith. If bandwith is a concern, you should use the app like normal, and avoid reloading. This can only be fully addressed by an installable app, which I have not written.
This app maintains a Websocket connection to notify you of updates in-game. To learn more about Websockets, click here


Elements

In this game, actions revolve around the 7 Elements
Each turn in a game, you will draw a card, and be able to create any Element you choose
Whenever you play a card, you will need to activate some Elements to do so
Each turn, any Element(s) chosen in previous turns will be made available again


Cards

This is a card game, and you will need virtual cards to play!
When you signup, you will receive 21 Packs of cards for free

Decks

Cards in this game belong to 1 of 7 Element types
When creating a deck, it can be helpful to put cards of the same type in a deck together
To edit a deck, click the edit button for the deck you wish to edit on the Decks screen. To go to the decks screen now, click here
